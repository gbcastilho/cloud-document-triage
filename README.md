# Cloud Document Triage

An N8N &amp; Python workflow to triage Google Drive documents using AI, OAuth2 and Postgres

## Overview

This project provides an automated workflow to triage documents stored in Google Drive. It leverages N8N for workflow automation, Python for AI-based document analysis, OAuth2 for secure authentication, and Postgres for data storage.

## Getting Started

To set up and run the project, you need to have [Docker](https://www.docker.com/) and [Docker Compose](https://docs.docker.com/compose/) installed.

Once you have them installed, run the following command:

```bash
docker compose up
```

This command will start all necessary services including N8N, Postgres, and any other dependencies defined in the Docker Compose file.
